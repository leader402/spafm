# 术语表（Glossary）

| 缩写 / 术语 | 全称 / 解释 |
|---|---|
| ST | Spatial Transcriptomics，空间转录组学 |
| scRNA-seq | Single-cell RNA sequencing |
| FM | Foundation Model，基础模型 |
| LLM | Large Language Model |
| LMM | Large Multimodal Model |
| Spot | ST 中带空间坐标的最小测序单元（Visium 直径 55µm） |
| Niche | 由空间相邻 spot/cell 组成的微环境 |
| H&E | Hematoxylin and Eosin staining，组织学染色 |
| MGM | Masked Gene Modeling，掩码基因建模（自监督目标） |
| NCL | Niche Contrastive Learning |
| GG-Recovery | Gene-Gene interaction Recovery |
| ST-Traj | Spatio-Temporal Trajectory pretraining |
| LoRA | Low-Rank Adaptation，低秩微调 |
| Deconvolution | 解卷积：从 spot 表达推断细胞类型组成 |
| Imputation | 插补：填补稀疏/缺失基因表达 |
| AnnData | 单细胞数据标准容器（Python） |
| Visium / Visium HD | 10x Genomics 测序型 ST 平台 |
| Xenium | 10x Genomics 影像型亚细胞 ST 平台 |
| MERFISH | Multiplexed Error-Robust FISH |
| Stereo-seq | BGI 的高分辨率 ST 平台 |
| ARI / AMI / ASW | 聚类评估指标（Adjusted Rand / Mutual Info / Avg Silhouette Width） |
| SSIM / RMSE / JS | 解卷积评估指标 |
