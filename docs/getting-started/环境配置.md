# 环境配置

## 系统要求

- Linux（推荐 Ubuntu 22.04 / Rocky 9）
- Python 3.10+
- CUDA 12.1+（如需 GPU 训练）
- 磁盘 ≥ 500 GB（数据语料阶段）

## 一键安装（conda / mamba）

```bash
# 1. 安装 mamba（如已有 conda 可跳过）
curl -L -O https://github.com/conda-forge/miniforge/releases/latest/download/Miniforge3-Linux-x86_64.sh
bash Miniforge3-Linux-x86_64.sh -b -p $HOME/miniforge3
source $HOME/miniforge3/bin/activate

# 2. 创建 SpaFM 环境
mamba env create -f environment.yml
conda activate spafm

# 3. 安装 PyTorch（按本机 CUDA 版本选择）
pip install torch torchvision --index-url https://download.pytorch.org/whl/cu121

# 4. 安装 SpaFM 本身（开发模式）
pip install -e ".[dev,dl]"

# 5. 验证
python -c "import spafm; print(spafm.__version__)"
make test
```

## 国内镜像（可选）

```bash
# pip
pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple

# conda
cat > ~/.condarc <<EOF
channels:
  - defaults
show_channel_urls: true
default_channels:
  - https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/main
  - https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free

  conda-forge: https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud
  bioconda: https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud
  pytorch: https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud
EOF
```

## 常见问题

| 问题 | 解决 |
|---|---|
| `ImportError: cannot import name 'AnnData'` | `pip install -U anndata>=0.10` |
| Squidpy 安装慢 | `mamba install -c conda-forge squidpy` |
| CUDA 不可用 | `nvidia-smi` 检查驱动；确认 PyTorch 与 CUDA 版本匹配 |
| `make lint` 提示 markdownlint 缺失 | `npm install -g markdownlint-cli`（可选，不强制）|
