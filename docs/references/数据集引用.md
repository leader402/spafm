# 数据集来源与引用（Datasets Citations）

本文件给 [docs/references/数据集清单.md](数据集清单.md) 中每条数据集补全
**论文 / DOI / 原始下载页 / BibTeX key**，论文撰写时直接对照引用。

> 字段说明：
>
> - **dataset_id**：与 [src/spafm/data/registry.py](../../src/spafm/data/registry.py) 一致
> - **paper**：首发论文标题（截短）
> - **DOI**：唯一标识
> - **portal**：原始数据下载入口页（不是 CDN 直链）
> - **bib key**：与 [datasets.bib](datasets.bib) 一一对应
>
> ⚠️ **使用许可**：所有 CC-BY-* 数据需在论文中按 license 要求注明来源；
> CC-BY-NC 数据**不可商用**；NanoString CosMx 是 custom license，需走申请流程。

## 3. 论文撰写引用规范

- **Methods 章节** ：cite 平台 paper（§1）+ 各预训练数据 paper（§2）
- **Data availability** ：列出 dataset_id + portal URL + DOI + license + 我们工作流脚本路径，例如：

  > "We pretrain SpaFM on N=… spatial transcriptomics slices aggregated from
  > publicly available sources, including HEST-1k [@jaume2024hest], the
  > spatialLIBD DLPFC cohort [@maynard2021dlpfc], 10x Genomics public Visium
  > demo slices, and the MOSTA mouse organogenesis atlas [@chen2022stereoseq].
  > Download scripts and dataset registry are available at
  > `scripts/data/` and `src/spafm/data/registry.py`."

- **License compliance** ：CC-BY-NC（MOSTA / ARTISTA / FlySta3D / Macaca / HEST-1k）所在切片仅用于学术评测，不进商业产品。
