# 数据集清单（Dataset Registry）

本文件登记 SpaFM 训练 / 评测使用的所有数据集，与 [src/spafm/data/registry.py](../../src/spafm/data/registry.py) 双向同步。

> 字段说明：
>
> - **Tier**：A=高质量带注释（评测用）；B=大规模未注释（预训练主力）；C=辅助/合成
> - **License**：使用前必须确认许可
>
> 📚 **论文引用 / DOI / BibTeX**：见 [数据集引用.md](数据集引用.md) 与 [datasets.bib](datasets.bib)。

## 增加新数据集流程

1. 在本文件添加一行，填齐 ID / 来源 / License
2. 在 [src/spafm/data/registry.py](../../src/spafm/data/registry.py) 中添加 `DatasetSpec`
3. 在 [scripts/data/](../../scripts/data/) 写下载脚本（可断点续传 + md5 校验）
4. 在 [src/spafm/data/loaders/](../../src/spafm/data/loaders/) 实现 `load(...)`，返回符合 schema 的 AnnData
5. 添加单测 `tests/data/test_<dataset>.py`（用极小 fixture）
6. 提交 PR：`feat: data - add <dataset_id>`
